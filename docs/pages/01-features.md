# Features and Navigation

## 🔗 Common Features and Screens

**Dynamic Breadcrumbs and Reload Bar**

Check your current location in the system at any moment, go back to previous pages or quickly refresh current page data.

![Dynamic Breadcrumbs and Reload Bar](/docs/images/v1.0/breadcrumb.png)

&nbsp;


**Login Page**

Access the application using your credentials or your Google account, or start the password recovery process if you have forgotten it.

![Login Page](/docs/images/v1.0/login.png)

&nbsp;


**Signup Page**

Create a new user account to start placing or tracking orders, checking prices, or posting product reviews.

![Signup Page](/docs/images/v1.0/signup.png)

&nbsp;


**Recover Password, Sent OTP, Verify Code and Reset Password Pages**

Regain access to your account by requesting a one-time password (OTP) sent to your registered email address. Enter the received code to verify your identity and, once confirmed, set a new application password.

![Recover Password Page](/docs/images/v1.0/recover-password.png)

![Sent OTP Page](/docs/images/v1.0/otp-sent.png)

![Verify OTP Code Page](/docs/images/v1.0/verify-otp-code.png)

![Reset Password Page](/docs/images/v1.0/reset-password.png)

&nbsp;


**Notifications Page and Notifications Tray**

Get real-time updates about all different events depending on your role: order updates, assignment change, new products offering, etc.

![Notifications Page](/docs/images/v1.0/notifications.png)

![Notifications Tray](/docs/images/v1.0/notifications-tray.png)

&nbsp;


**Connection Status Section and Online Indicators**

Check current system status, last session insights (date, time and duration) and other users connection status (online / offline) across the app.

![Connection Status Section](/docs/images/v1.0/connection-section.png)

![Connection Status Indicators](/docs/images/v1.0/connection-indicator.png)

&nbsp;
