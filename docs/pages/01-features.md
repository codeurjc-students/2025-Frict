# Features and Navigation

## 🔗 Common Features and Screens

**Dynamic Breadcrumbs and Reload Bar**

Check your current location in the system at any moment, go back to previous pages or quickly refresh current page data.

![Dynamic Breadcrumbs and Reload Bar](/docs/images/v0.2/breadcrumb-reload.png)

&nbsp;


**Login Page**

Access the application using your credentials or your Google account, or start the password recovery process if you have forgotten it.

![Login Page](/docs/images/v0.1/login.png)

&nbsp;


**Recover Password, Sent OTP and Reset Password Pages**

Regain access to your account by entering a one-time password (OTP) sent to your registered email address. Once verified, update your application password.

![Recover Password Page](/docs/images/v0.1/recover-password.png)

![Sent OTP Page](/docs/images/v0.1/otp-sent.png)

![Reset Password Page](/docs/images/v0.1/reset-password.png)

&nbsp;


**Notifications Page and Notifications Tray**

Get real-time updates about all different events depending on your role: order updates, assignment change, new products offering, etc.

![Notifications Page](/docs/images/v0.2/notifications.png)

![Notifications Tray](/docs/images/v0.2/notifications-tray.png)

&nbsp;


**Connection Status Section and Online Indicators**

Check current system status, last session insights (date, time and duration) and other users connection status (online / offline) across the app.

![Connection Status Section](/docs/images/v0.2/connection-section.png)

![Connection Status Indicators](/docs/images/v0.2/connection-indicator.png)

&nbsp;
